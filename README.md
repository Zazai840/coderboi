# Coderboi

This repo contains the core components of Coderboi, an AI coding agent.

## What it does:
Coderboi is designed to interact with a codebase, understand user requests, and perform actions such as listing files, reading file contents, writing files, and executing Python scripts. It uses a set of predefined tools to achieve these tasks.

## How it works:
Coderboi processes user prompts, analyzes the request, and then utilizes its available tools (e.g., `get_files_info`, `get_file_content`, `write_file`, `run_python_file`) to fulfill the request. It plans its actions, executes them, and provides responses based on the outcomes.
